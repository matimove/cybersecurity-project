# Cybersecurity project (OWASP 2021)

This is a project made for the Cyber Security Base: Course Project I course. 

The idea is to build a webapp using Django and Python, introduce five known vulnerabilities from the OWASP 2021 list, and show how they can be fixed.

The webapp was based on the official Django tutorial: "Writing your first Django app"

Five vulnerabilities chosen from the OWASP 2021 list were:
Broken Access Control (A01)
Cryptographic Failures (A02)
SQL Injection (A03)
Security Misconfiguration (A05)
Authentication Failures (A07)
